생활코딩 - Web1

기초 tag와 web 기본 구조

1. strong : 굵은 글씨
2. u: u : 밑 (underline)
3. br: br : 줄바꿈 (create a line break in a text.)
4. li 목차 (list)
* ol (ordered list) : 목차 번호를 자동으로 입력 (부모 태그)
* ul (unordered list) : 번호 없 (부모태그)
5. p: 단락 (paragraph) : 앞 뒤로 닫아야함
6. h1~h6: 제목 (heading)
7. img: 이미지 (image)
< > 안에 img, src="이미지 제목", width= 페이지에 맞출 크기 순으로 입력
8. a: anchor 앵커의 약자. 정보의 바다에 닻을 내린다 = 링크를 거는 역할로 href와 함께 쓰인다.
*href : (hypertext reference)
*링크를 걸 글씨 앞 뒤로 a를 걸고, 앞의 a에 띄어쓰기 후 href="링크"
*target="_blank" 는 새 창으로 열기
